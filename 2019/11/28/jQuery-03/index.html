jQuery补充知识点

隐式迭代

基本概念

隐式迭代：jQuery在设置属性时会自动的遍历，因此我们不需要再遍历

  1. jQuery在执行设置性操作时，会给所有的元素都设置上相同的值。
  2. jQuery在执行获取性操作时，只会返回第一个元素对应的值。
  3. 如果想要给每一个元素都设置不同的值，需要手动进行遍历jQuery对象。

链式编程

链式编程的原理：设置性操作会返回一个jQuery对象，因此可以继续调用jQuery的方法。

  1. 设置操作的时候，可以使用链式编程。
  2. 获取操作的时候，无法使用链式编程。
